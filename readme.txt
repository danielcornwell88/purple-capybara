netscan.py
Quickly scan a network and generate a pdf report based on supplied arguments

gitcheck.py
Provide a github url to scan and provide a report based on supplied arguements

report.py
Support functions to generate pdf reports
Based on:
Stars
Forks
Last commit data relative to current date
Number of unique contributors
Lines of code and frequency of each contributor 
Commit history 
Languages used in project 
Number of open and closed issues
Number of pull requests
Number of security advisors / policy etc
Dependencies and reputation of dependencies 
Finally, the score is compiled to provide a score. Higher means a more active project.
